# Trainer - Watson-Glaser Style Critical Thinking Interface

A web-based training interface for critical thinking skills based on Watson-Glaser methodology.

## Quick Start

```bash
cd trainer
npm install
npm start
```

Then open http://localhost:8080/advanced.html

## Files

- `advanced.html` - Main training interface
- `four_layer.html` - Four-layer reasoning view  
- `autonomous.html` - Autonomous mode
- `agent_profiles.js` - Agent persona configurations

## Testing

```bash
npm test
```
